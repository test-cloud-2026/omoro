# OMORO Visual Dictionary（おもろ図鑑）

> ランダムなお題に画像と名前をつけて、自分だけの不思議な図鑑を作るアプリ

---

## 🌐 デモ（実際に使えます）

👉 https://omoro-lac.vercel.app/

---

## 🧭 このアプリでできること

「存在しないのに、ありそうなもの」をテーマに、
お題に沿って画像と名前を登録し、自分だけの図鑑を作ることができます。

- お題に対して写真を当てはめる
- 名前をつけて“存在”を定義する
- URLで保存・共有する

👉 創作 × 収集 × 表現 を組み合わせた体験型アプリです

---

## 👤 想定ユーザー

- 創作が好きな人（高校生・学生など）
- 写真やイラストを使って遊びたい人
- SNSでシェアしたくなるコンテンツを作りたい人

---

## ✨ 主な機能

- **お題生成**

  - ホラー / 日常 / SF / 乙女 の4カテゴリ
  - ランダムで6つのお題を表示

- **画像アップロード**

  - 各カードに画像を登録（最大5MB）
  - Supabase Storage に保存

- **名前入力**

  - お題に対して自由に名前をつけられる

- **保存 & URL共有**

  - 保存すると専用URLが発行
  - 同じ内容を再表示可能

- **スクショモード**

  - 図鑑風の見た目に切り替え
  - SNS投稿用に最適化

- **リセット機能**
  - 入力内容を一括クリア

---

## 🛠 使用技術

| 分類           | 技術                              |
| -------------- | --------------------------------- |
| フロントエンド | HTML / CSS / JavaScript（バニラ） |
| UI             | Swiper.js                         |
| バックエンド   | Supabase（PostgreSQL）            |
| ストレージ     | Supabase Storage                  |
| セキュリティ   | Row Level Security（RLS）         |
| デプロイ       | Vercel                            |

---

## 🚀 使い方

1. カテゴリを選択（ランダムも可）
2. 表示されたお題に画像をアップロード
3. 名前を入力
4. 「保存」ボタンでURLを発行

👉 URLを開くと同じ図鑑が再現されます

---

## 🔒 セキュリティ設計（ポイント）

- Storageは**privateバケット**
- パスを `works/<UUID>/<topicId>.png` に制限
- RLSでアップロード制御
- 将来的に user_id ベースの制御へ拡張可能

---

## 💡 工夫した点

- **1ファイル構成**

  - 環境構築なしで動作

- **図鑑UI設計**

  - 通し番号 / カード構成 / 余白設計

- **URL復元機能**

  - `?work=ID` で状態を完全再現

- **Storage設計**
  - Base64からURL保存へ移行（軽量化＆実用性）

# OMORO Visual Dictionary（おもろ図鑑）

> ランダムなお題に画像と名前をつけて、自分だけの不思議な図鑑を作ろう。

---

## アプリ概要

テーマ（お題）がランダムに出題され、それに合う画像をアップロードして名前をつけるだけで、世界にひとつだけのビジュアル図鑑が完成します。  
作った図鑑はURLで保存・共有できます。

## 主な機能

- **お題のランダム生成** — ホラー / 日常 / SF / 乙女 の4カテゴリから6枚のお題を出題
- **画像アップロード** — お題ごとに写真や画像をアップロードしてビジュアル図鑑を作成（上限5MB）
- **名前をつける** — 自分が見つけた"それ"に、オリジナルの名前をつけられる
- **URL共有** — 保存するとURLが発行され、SNSやLINEでかんたんにシェアできる
- **スクショモード** — アンティーク図鑑風のデザインに切り替えてスクリーンショット映えする表示に変換
- **カテゴリフィルター** — 好きなカテゴリだけでお題を絞り込んで生成
- **リセット機能** — 入力した名前・画像をまとめてクリアしてやり直せる

## 使用技術

| 分類 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（バニラ・ビルドツールなし） |
| UIライブラリ | [Swiper.js](https://swiperjs.com/)（カードのカルーセル表示） |
| データベース | Supabase（PostgreSQL + Row Level Security） |
| ファイルストレージ | Supabase Storage（画像の保存・配信） |
| デプロイ | 未設定（単一HTMLファイルで動作） |

## 使い方

**1. カテゴリを選ぶ**  
「ランダム」「ホラー」「日常」「SF」「乙女」からひとつ選ぶと、6つのお題が表示されます。

**2. 画像をアップロードして名前をつける**  
各カードの画像エリアをタップして写真を選択。テキスト欄にオリジナルの名前を入力します。

**3. 保存してシェア**  
「この作品を保存する」ボタンを押すと専用URLが発行されます。X（旧Twitter）やLINEでシェアできます。

## 工夫した点

- **シングルファイル構成**  
  `index.html` 1ファイルにHTML・CSS・JavaScriptをすべて収めており、環境構築なしにブラウザで動きます。

- **スクショモードのデザイン**  
  アンティーク図鑑風のテクスチャ・明朝体・通し番号（No.01〜06）を組み合わせ、SNSに映える見た目を実装しました。

- **Supabase Storage のセキュリティ設計**  
  アップロード先のパスを `works/<UUID>/<topicId>.png` 形式のみ許可するRLSポリシーを設定し、過剰アップロードを防いでいます。MIMEタイプ・ファイルサイズの制限もバケット設定で二重に管理しています。

- **URL共有と作品の復元**  
  `?work=ID` のURLを読み込むと、保存済みの作品（お題・名前・画像）を自動で復元して表示します。
